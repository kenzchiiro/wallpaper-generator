# Wallpaper Generator

A minimal, browser-based wallpaper generator. Design custom wallpapers with gradients, typography, noise textures, and geometric shapes — then export at any resolution.

**[Try it live →](https://kenzchiiro.github.io/wallpaper-generator/)**

## Features

- Gradient, solid color & texture backgrounds
- Custom text with font controls
- Geometric shape overlays
- Noise / grain effects
- Export at custom resolution (PNG)

## Usage

Open `index.html` in any modern browser — no install needed.
